## Getting Started

### Accessing the Platform

1. **Visit the Website**: Open your web browser and go to [bharatsahayak.in](https://bharatsahayak.in)
2. **No Registration Required**: You can start using the platform immediately without creating an account
3. **Choose Your Language**: Select your preferred language from the language selector (11 languages available)

### System Requirements

**Supported Browsers**:
- Google Chrome (recommended)
- Mozilla Firefox
- Safari
- Microsoft Edge

**Minimum Requirements**:
- Internet connection (works on 2G/3G/4G/WiFi)
- Modern web browser (updated within last 2 years)
- JavaScript enabled

**For Voice Features**:
- Microphone access (for voice input)
- Speaker or headphones (for voice output)

### First Time Setup

1. **Select Language**: Click the language selector in the top-right corner
2. **Enable Permissions**: If using voice features, allow microphone access when prompted
3. **Start Chatting**: Click "Start Chatting" on the landing page

## Using the Chat Interface

### Starting a Conversation

1. **Type Your Question**: Click the text input box at the bottom
2. **Be Specific**: Describe your situation or what you're looking for
   - Example: "I am a farmer with 1 hectare of land. What schemes can help me?"
3. **Send Message**: Press Enter or click the send button

### Example Questions

**Finding Schemes**:
- "What education schemes are available for my daughter?"
- "I need financial help for medical treatment"
- "Show me schemes for small business owners"

**Getting Details**:
- "Tell me more about PM-KISAN"
- "How do I apply for Ayushman Bharat?"
- "What documents do I need for this scheme?"

**Checking Eligibility**:
- "Am I eligible for this scheme?"
- "What are the income requirements?"
- "Can women apply for this scheme?"

### Understanding Responses

**Scheme Cards**: When the assistant recommends schemes, you'll see cards with:
- **Scheme Name**: Official name of the program
- **Description**: What the scheme provides
- **Eligibility Summary**: Quick overview of who can apply
- **Application Steps**: How to apply
- **Check Eligibility Button**: Detailed eligibility check

**Text Responses**: The assistant provides explanations in your language with:
- Clear, simple language
- Step-by-step instructions
- Links to official websites
- Contact information when available

### Conversation Tips

✅ **Do**:
- Ask follow-up questions
- Request clarifications
- Describe your specific situation
- Mention your state, occupation, or other relevant details

❌ **Don't**:
- Share sensitive personal information (Aadhaar number, bank details)
- Expect the assistant to process applications
- Assume information is always up-to-date (verify on official websites)

## Voice Interaction

### Using Voice Input

1. **Click Microphone Button**: Located next to the text input
2. **Allow Microphone Access**: Grant permission if prompted
3. **Speak Clearly**: Say your question in any supported language
4. **Watch the Waveform**: Visual feedback shows you're being recorded
5. **Automatic Stop**: Recording stops after 2 seconds of silence
6. **Manual Stop**: Click the microphone button again to stop

**Tips for Better Recognition**:
- Speak clearly and at a normal pace
- Minimize background noise
- Hold device 6-12 inches from your mouth
- Speak in complete sentences

### Using Voice Output

1. **Click Speaker Button**: Located on assistant messages
2. **Listen to Response**: Audio plays automatically
3. **Pause/Resume**: Click button again to control playback
4. **Adjust Volume**: Use your device's volume controls

**Voice Output Features**:
- Natural-sounding voices for each language
- Adjustable playback speed (coming soon)
- Works offline after initial load (coming soon)

### Troubleshooting Voice Features

**Microphone Not Working**:
- Check browser permissions (Settings → Privacy → Microphone)
- Ensure microphone is not muted
- Try a different browser
- Check if another app is using the microphone

**Poor Transcription Quality**:
- Reduce background noise
- Speak more clearly
- Try typing instead for complex terms
- Check internet connection

**Audio Not Playing**:
- Check device volume
- Ensure speakers/headphones are connected
- Try refreshing the page
- Check browser audio permissions

## Checking Eligibility

### Step-by-Step Process

1. **Find a Scheme**: Browse or ask the assistant for recommendations
2. **Click "Check Eligibility"**: On the scheme card
3. **Fill the Form**: Provide requested information
   - Age
   - Gender
   - Annual income
   - State of residence
   - Category (General/OBC/SC/ST)
   - Occupation
   - Other scheme-specific details
4. **Submit**: Click "Check Eligibility" button
5. **View Results**: See if you're eligible with detailed explanation

### Understanding Eligibility Results

**Eligible** ✅:
- Green checkmark indicator
- List of criteria you meet
- Benefits you'll receive
- Application process steps
- Required documents
- Link to official website

**Not Eligible** ❌:
- Red cross indicator
- Explanation of why you don't qualify
- Specific criteria not met
- Suggestions for alternative schemes
- Tips on what might make you eligible in the future

### Privacy Note

Your eligibility information is:
- Stored temporarily (24 hours only)
- Not shared with third parties
- Used only for eligibility checking
- Automatically deleted after your session

## Accessibility Features

### For Screen Reader Users

**Supported Screen Readers**:
- NVDA (Windows)
- JAWS (Windows)
- VoiceOver (macOS, iOS)
- TalkBack (Android)

**Navigation**:
- Use Tab key to move between elements
- Use Enter/Space to activate buttons
- Use arrow keys in lists and menus
- Headings are properly structured (H1, H2, H3)

**Announcements**:
- New messages announced automatically
- Eligibility results read aloud
- Error messages clearly communicated
- Loading states indicated

### Keyboard Navigation

**Essential Shortcuts**:
- `Tab`: Move to next element
- `Shift + Tab`: Move to previous element
- `Enter`: Activate button or link
- `Space`: Toggle checkboxes
- `Esc`: Close modals or dialogs
- `/`: Focus on search box

**Skip Links**:
- Press `Tab` on page load to reveal "Skip to main content" link
- Bypass navigation and go straight to content

### Visual Accessibility

**Dark Mode**:
1. Click theme toggle (sun/moon icon)
2. Reduces eye strain in low light
3. Maintains proper contrast ratios

**Text Sizing**:
- Use browser zoom (Ctrl/Cmd + Plus/Minus)
- Layout adapts up to 200% zoom
- Text remains readable

**Color Contrast**:
- All text meets WCAG AA standards (4.5:1 ratio)
- Important elements have 3:1 contrast
- Color is not the only indicator

### Low Bandwidth Mode

**When to Use**:
- Slow internet connection (2G/3G)
- Limited data plan
- Rural areas with poor connectivity

**How to Enable**:
1. Click settings icon
2. Toggle "Low Bandwidth Mode"
3. Or automatically suggested when slow connection detected

**What Changes**:
- Animations disabled
- Images compressed (50% quality)
- Audio compressed (70% smaller)
- Lazy loading for non-critical content
- Simplified UI elements

## Troubleshooting

### Common Issues

#### 1. Page Not Loading

**Symptoms**: Blank page, loading spinner doesn't stop

**Solutions**:
- Refresh the page (Ctrl/Cmd + R)
- Clear browser cache and cookies
- Try a different browser
- Check internet connection
- Disable browser extensions temporarily

#### 2. Chat Not Responding

**Symptoms**: Messages sent but no response

**Solutions**:
- Wait 10-15 seconds (AI processing takes time)
- Check internet connection
- Refresh the page
- Try sending a simpler message
- Clear session and start new conversation

#### 3. Voice Input Not Working

**Symptoms**: Microphone button doesn't work, no transcription

**Solutions**:
- Check browser microphone permissions
- Ensure microphone is not muted
- Close other apps using microphone
- Try a different browser
- Use text input as alternative

#### 4. Incorrect Language Detection

**Symptoms**: Wrong language detected for voice/text

**Solutions**:
- Manually select language from dropdown
- Speak/type more clearly
- Use language-specific keyboard
- Refresh page and try again

#### 5. Eligibility Form Not Submitting

**Symptoms**: Form doesn't submit, error messages

**Solutions**:
- Fill all required fields (marked with *)
- Check for validation errors (red text)
- Ensure values are in correct format
- Try refreshing and filling again
- Use different browser

#### 6. Scheme Cards Not Displaying

**Symptoms**: No schemes shown after query

**Solutions**:
- Try rephrasing your question
- Be more specific about your needs
- Check if schemes exist for your category
- Refresh page and try again

### Error Messages

**"Session Expired"**:
- Your session lasted more than 24 hours
- Click "Start New Session" to continue

**"Network Error"**:
- Check internet connection
- Try again in a few moments
- Contact support if persists

**"Service Unavailable"**:
- Temporary server issue
- Try again in 5-10 minutes
- Check status page for updates

**"Rate Limit Exceeded"**:
- Too many requests in short time
- Wait 1 minute and try again
- Avoid rapid clicking

### Browser-Specific Issues

**Chrome**:
- Clear cache: Settings → Privacy → Clear browsing data
- Reset permissions: Settings → Privacy → Site settings

**Firefox**:
- Clear cache: Options → Privacy → Clear Data
- Reset permissions: Options → Privacy → Permissions

**Safari**:
- Clear cache: Safari → Clear History
- Reset permissions: Safari → Preferences → Websites

**Edge**:
- Clear cache: Settings → Privacy → Clear browsing data
- Reset permissions: Settings → Site permissions
